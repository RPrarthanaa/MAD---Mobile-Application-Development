# Syllabus

This unit covers software development principles and best practices for designing, implementing, and testing mobile applications. Topics include:

* Mobile user interface design
* Structural design using standard mobile development frameworks and APIs
* Integration with online services
* Data storage and persistence using Room
* Evaluating security and best coding practices
* Testing methodologies and practices for mobile apps

---

# Assignment 1

Developed a Connect 4 mobile game application

## Features:
- [ ] Two-player gameplay mode & AI opponent mode
- [ ] Customizable player profiles and board themes
- [ ] Undo move feature
- [ ] Score tracking with statistics display
- [ ] State persistence despite orientation changes

# Assignment 2

Developed a mobile application to track daily calorie intake

## Features:
- [ ] Set and track daily calorie goals
- [ ] Add, edit, and manage meal entries for each day
- [ ] Search food items using an external API to retrieve calorie information if unknown
- [ ] View previous meal records
- [ ] Store data locally
- [ ] Cloud storage for meal photos
- [ ] State persistence despite orientation changes
